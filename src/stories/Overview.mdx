# Predusk AI - Component Documentation

Welcome to the Predusk AI component library documentation. This project is built with Next.js, Tailwind CSS, and shadcn/ui components.

## Project Overview

Predusk AI is a modern AI dashboard application featuring:

- **Modern UI Components** - Built with shadcn/ui and Radix UI primitives
- **Responsive Design** - Fully responsive across all device sizes
- **TypeScript** - Full type safety throughout the application
- **Storybook Integration** - Component development and documentation

## Tech Stack

- **Framework**: Next.js 15 with App Router
- **Styling**: Tailwind CSS with CSS-in-JS
- **Components**: shadcn/ui + Radix UI
- **Development**: Storybook for component development
- **Deployment**: Netlify with automatic deployments

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Run Storybook:
   ```bash
   npm run storybook
   ```

## Component Library

This Storybook contains interactive examples of all UI components used in the Predusk AI application. Each component is documented with:

- **Props API** - All available properties and their types
- **Usage Examples** - Different states and variations
- **Accessibility** - ARIA compliance and keyboard navigation
- **Design Tokens** - Color, typography, and spacing guidelines

## Key Components

### Layout Components
- **ChatArea** - Main chat interface with message handling
- **Sidebar** - Navigation and user controls
- **Navigation** - Top navigation bar with theme toggle

### UI Components
- **Button** - Primary, secondary, and icon button variants
- **Card** - Content containers with headers and actions
- **Select** - Dropdown selection with search
- **Textarea** - Multi-line text input with validation
